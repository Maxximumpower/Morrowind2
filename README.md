#Morrowind 2

Morrowind II contains several of the most beloved mods of 1.12.2:
Thaumcraft
Oh The Biomes You'll Go
Tinker's Construct
Astral Sorcery
Recurrent Complex
Baubles
Ice and Fire
Encanting Plus
And 80+ more mods!
